Basics

you will need to put some elements in most html code,such as !DOCTYPE html to announce the type of file it is, to announce you want to have yout html element in english use html lang=en.

Remember to close your tags unless your tag is self-closing

Remember to link your stylesheet to your page with a link element typically followed by rel="stylesheet" href="styles.css".

Try to leave a gap between elements you are going to nest many other elements in so you are less likely to nest them outside the parent.

If not obvious, separating large sections of a page should be done with section elements, with child elements to be nested inside.

Html Essentials

knowing your header numbers can help you have font sizing without css with h elements, h elements go from 1-6 with 1 being the biggest and 6 being the smallest.

use Head elements to nest title elements to give your webpage a title on the navigation bar.

  • META tags with the attributes name="viewport" content="width=device-width, initial scale=1.0" will be the size of the device viewing it
  • most webpages use UTF-8
  • Nav elements can be used for links to take you to other pages or other sections within a page you are currently viewing.

Make sure you assign classes and ids to your elements to make selections with css much easier.

CSS Styling

you can make all kinds of things with css, boxes, dividers, borders, if you can imagine it and figure ou the dimensions you can reasonably make them with flexbox.

using css you can style webpages and text and even orient text on the webpage to wherever you may want.

common css variables are color: (whatever color you want); to color the text on the screen background-color: (your preferred color) to change the color of the background and text-align: (top,left, right, bottom); to align the text to different sides of the screen.
